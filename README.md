# Flask_Socketio_Server
使用flask_socketio构建的服务端应用

# 配置  

```shell

$ pip install Flask

$ pip install flask-socketio

$ pip install eventlet

$ pip install gunicorn # 要求 python 3.x >= 3.4

```

# 运行

在服务器上执行 start.sh 文件

```shell
sh ./start.sh
```
